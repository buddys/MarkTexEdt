This folder contains the design assemblies, used to provide design-time features
for the Awesomium.NET components, in Visual Studio 2010 & Visual Studio 2010.

THESE ASSEMBLIES ARE NOT REDISTRIBUTABLE.

If tou have used the Windows Instaler to install the SDK, these assemblies
must already be in your system's GAC. Otherwise, you can use the Setup utility
under the Assemblies folder. For details, read the README.txt under the
Assemblies folder.
